## Getting Started

<p>
</p>

### Prerequisites

[Go](https://go.dev/dl/)
Must be version at least 1.22.0

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/RedconOne/jobby.git
   ```

1. Run the init file to create mandatory excel spreadsheet for data storage
   ```sh
   go run init.go
   ```
